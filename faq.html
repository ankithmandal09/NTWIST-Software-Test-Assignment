Frequently Asked Questions

What technologies were used to build this website?

This website was built using only HTML, CSS, and vanilla JavaScript as required by the assignment. No external libraries or frameworks were used.

How is the form validation implemented?

The form validation uses a combination of HTML5 validation attributes (required, pattern, type) and custom JavaScript validation for enhanced user experience.

How does the sortable list work?

The sortable list is implemented using JavaScript DOM manipulation. When the sort button is clicked, JavaScript sorts the list items alphabetically without reloading the page.

Is this FAQ accordion using JavaScript?

No! This FAQ accordion is implemented using only HTML and CSS. It uses the HTML5 <details> and <summary> elements which provide native expand/collapse functionality without any JavaScript.

How is the star rating implemented?

The star rating is implemented using radio buttons with custom CSS styling. When a user clicks on a star, the corresponding radio button is selected, and styling is applied to show the selected rating.
